# lag-clothing
LAG Clothing

## Project Overview

LAG Clothing is a jersey-based clothing brand website currently under development using **Flutter Web** and **Firebase**. The goal of this project is to build a modern, responsive, and scalable e-commerce platform for showcasing and selling premium jerseys.

This repository contains the source code for the website and will be updated as development progresses.

---

## Tech Stack

- **Frontend:** Flutter (Dart)
- **Backend:** Firebase
- **Database:** Cloud Firestore
- **Authentication:** Firebase Authentication
- **Hosting:** Firebase Hosting
- **Version Control:** Git & GitHub
- **IDE:** Visual Studio Code
